# IPL Data Analysis 🏏

## Overview

This project analyzes IPL match and ball-by-ball data using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## Objectives

* Analyze team performance
* Identify Orange Cap and Purple Cap winners
* Study dismissal patterns
* Evaluate toss impact
* Explore season-wise scoring trends

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Insights

* Virat Kohli is the highest run scorer in IPL history.
* Yuzvendra Chahal is the leading wicket taker.
* Mumbai Indians is the most successful IPL franchise.
* Toss has minimal impact on match outcomes.
* IPL has become increasingly batting-friendly over time.

* Orange Cap Analysis
* Purple Cap Analysis
* Runs by Season
* Team Performance Heatmap
